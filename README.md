Pebble-SmoothSquare
===================

Pebble watchface where the digits animate off and on the screen.

You can download the pbw from our friends at MyPebbleFaces.com

http://www.mypebblefaces.com/view?fID=743&aName=andrewholmes&pageTitle=SmoothSquare+%28animated%29&auID=994

If you like the face, or the sourcecode is useful please leave on donation for the MyPebbleFaces website which helps fund their servers.  Anything left over goes to charity.
